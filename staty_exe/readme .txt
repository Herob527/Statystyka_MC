======= Ogólne informacje =======
Próba dla każdego materiału : 200 kilofów.
Ustalona precyzja na wyjściu: Do części tysięcznej.
Kombinowanie z folderami w tym folderze sprawi, że program nie będzie działał (nie zaimplementowałem jeszcze tego przypadku)
Żeby strona działała poprawnie, wstawiona w foldery, gdzie działa serwer HTTP (gdzie się odnosimy).
np. XAMPP, przenosimy plik do "htdocs" i odnosimy się do niego wpisując "localhost/[nazwaPliku]"

======= Dla programistów =======
Na Linux (w tym na MacOS) program nie zadziała, bo wykorzystuje komendy w systemie Windows.
Program tworzony w Visual Studio C++ 2019.
Program nie sprawdza, czy moda faktycznie jest modą (tzn. czy wszystkich wartości jest tyle samo). W fazie rozważań.
